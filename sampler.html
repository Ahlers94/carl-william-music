Subsystem Details

Welcome to the μS-1000 micro-sampling deck. Record incoming microphone or live hardware line audio streams straight onto any memory node array block, fine-tune the tracking markers inside the trim buffer, and assign notes out to the polyphonic synthesiser keyboard engines or connected class-compliant MIDI equipment.

Signal Routing

The internal signal array travels through an isolated Envelope Gate engine directly into a variable Bucket Brigade Delay line processing link, feeding into an output attenuator node running inside standard Web Audio contexts.
